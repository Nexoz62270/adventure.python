# ⚔️ Adventure.py — Shadow of the Dragon 🐉

A text-based RPG adventure game built in Python. Fight monsters, explore zones, level up and defeat the Ancient Dragon to save the kingdom!

---

## 🎮 Features

- ⚔️ **Turn-based combat** with HP bars
- 🗺️ **4 unique zones** — Dark Forest, Village, Ancient Ruins, Dragon's Castle
- 🛒 **Shop system** — buy potions, upgrade your sword and shield
- 📈 **Leveling system** — gain XP and level up after each fight
- 🌍 **FR / EN language selection** at startup (English by default)
- 🎮 **Secret cheat panel** — press `C` anytime (password protected 🔒)

---

## 🚀 How to play

### Requirements
- Python 3.x — download at [python.org](https://www.python.org)

### Run the game
```bash
python adventure.py
```

Or on Mac/Linux:
```bash
python3 adventure.py
```
